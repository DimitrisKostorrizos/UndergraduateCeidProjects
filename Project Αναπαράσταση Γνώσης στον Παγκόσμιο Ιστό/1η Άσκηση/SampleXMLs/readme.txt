 
--------------------------------------------------------------------------------------------
Τα αρχεία αυτά σας δίνονται για να ελεγξετε τα XML σχήματα σε κάθε ερώτημα της άσκησης.

Ο αριθμός μπροστά απο το αρχείο αντιστοιχεί στο ερώτημα της άσκησης που αναφέρεται.

Η ένδειξη RIGHT σημαίνει οτι το αρχείο αυτό πρέπει να γίνεται validate χωρίς Errors 
απο το σχήμα που φτιάξατε.

Η ένδειξη WRONG σημαίνει οτι το αρχείο περιέχει κάποιο λάθος και αν έχετε φτιάξει σωστό 
XML Schema θα πρέπει να μην γίνεται επιτυχώς validation.

Για παράδειγμα το αρχείο "5_RIGHT_schedule.xml" θα δοκιμάσετε να το κάνετε validate αφού
φτιάξετε το XML SChema που απαντάει στο ερώτημα 5 και αν το σχήμα είναι σωστό 
τότε θα πρέπει το validation να γίνει χωρίς errors.

---------------------------------------------------------------------------------------------

Το αρχείο 10_schedule.xml θα το χρησιμοποιήσετε στο ερώτημα 10 προκειμένου να δοκιμάσετε 
τα XPath ερωτήματα που θα φτιάξετε και στο ερώτημα 11 για να εφαρμόσετε σε αυτό το αρχείο 
μετασχηματισμού XML (XSL).
